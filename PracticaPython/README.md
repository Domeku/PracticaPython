# Proyecto de Analisis de Sentimiento Simple (NLP)

### Descripcion
Este es un pequeño proyecto de procesamiento de lenguaje natural que identifica si una frase es positiva, negativa o neutral. Realizado para la Practica 2.

### Version de Python

### Instalacion y Ejecucion
1. Clonar el repositorio.
2. Crear y activar entorno virtual:
   - `python -m venv venv`
   - `source venv/bin/activate` (o `venv\Scripts\activate` en Windows)
3. Instalar dependencias:
   - `pip install -r requirements.txt`
4. Ejecutar el proyecto:
   - `python main.py`
